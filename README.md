# Trabalho-Modelo-API
Atividade Avaliativa da Disciplina de Fundamentos de Inteligência Artificial

## Como Rodar ##
1. Rode o comando npm install nas duas pastas, Front e cnn-api-aulas
2. Na pasta cnn-api-aulas rode o comando npm start
3. Na pasta Front rode o comando npm run dev, pressione a tecla 'o' para abrir no navegador

## Como Usar ##
1. Baixe uma imagem de uma arma medieval da sua escolha entre espadas de uma mão, de duas mãos, machados, maças, lanças e alabardas (Ou utilizada a imagem teste.jpeg presente na pasta cnn-api-aulas, que representa uma machado)
2. Na tela aberta no passo 3 do "como rodar" clique no botão "Present Weapon" para abrir a submissão de imagem
3. Clique em "Procurar... Nenhum Arquivo" para selecionar a imagem que você baixou
4. Clique em "Submit" para ver a (decepção) mágica acontecer
